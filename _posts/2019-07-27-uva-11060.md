# Uva 11060-Beverages
### 難度 **
### 解法
跟[Uva10305](/2019/07/29/uva-10305/)一樣的問題只是輸入是字串,為了提升效能(HashMap的put/get接近O(1)ArrayList的indexOf要O(n))用map分別記錄飲品名稱跟編號,其他解法一樣,因為Input空行跟題目講的不一樣RuntimeError了一次zz

### 程式碼

``` java=
public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringBuilder sb = new StringBuilder();
		String str;
		int caseIndex = 1;
		while ((str = br.readLine()) != null && str.length() != 0) {
			int N = Integer.parseInt(str.trim());
			int[][] orders = new int[N][N];
			int[] toal = new int[N];
			Map<String, Integer> beverages = new HashMap<>();
			Map<Integer, String> numbers = new HashMap<>();
			for (int i = 0; i < N; i++) {
				String beverage = br.readLine();
				beverages.put(beverage, i);
				numbers.put(i, beverage);
			}

			int M = Integer.parseInt(br.readLine().trim());
			for (int i = 0; i < M; i++) {
				String[] order = br.readLine().split(" ");
				int B1 = beverages.get(order[0]);
				int B2 = beverages.get(order[1]);
				if (orders[B1][B2] != 1) {
					orders[B1][B2] = 1;
					toal[B2]++;
				}
			}
			sb.append("Case #" + caseIndex++ + ": Dilbert should drink beverages in this order:");
			ArrayList<Integer> order = topological(orders, toal);
			order.stream().forEach(a -> sb.append(" " + numbers.get(a)));
			sb.append(".\n\n");
			br.readLine();
		}
		System.out.print(sb);
	}

	private static ArrayList<Integer> topological(int[][] orders, int[] toal) {
		ArrayList<Integer> order = new ArrayList<>();
		int N = toal.length;
		for (int i = 0; i < N; i++) {
			int index = 0;
			while (toal[index] != 0) {
				index++;
			}
			toal[index] = -1;
			order.add(index);
			for (int x = 0; x < N; x++) {
				toal[x] -= orders[index][x];
			}
		}

		return order;
	}
}
```
